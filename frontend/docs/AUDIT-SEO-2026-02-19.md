## PLAN D'ACTIONS PRIORISEES

### P0 — CRITIQUE (Impact SEO majeur, a faire cette semaine)

#### 1. Header navigation : remplacer `<button>` par `<Link>`
- **Impact :** Les liens principaux (Galerie, Actualites, Reservation) sont invisibles pour Googlebot
- **Fichiers :** `HeaderVariants.tsx` L106-120, `MobileMenu.tsx` L63-91
- **Effort :** 1h
- **Gain :** Les pages cles deviennent decouvrables depuis la navigation principale

#### 2. Blog slugs cross-locale : 14 articles retournent 404 en non-FR
- **Impact :** 70 URLs en 404 (14 articles x 5 locales), hreflang casse
- **Cause :** `generateStaticParams` utilise uniquement les slugs FR, mais 14 posts ont des slugs differents (-2) dans les autres locales
- **Fichiers :** `[locale]/actualites/[slug]/page.tsx`, `src/lib/posts.ts`
- **Effort :** 2h
- **Solution :** Implementer une resolution slug locale-aware dans `getPost()` et generer les static params par locale

#### 3. Boilerplate duplique dans 11 articles blog + liens admin
- **Impact :** Contenu duplique sur 11 posts, liens vers admin.bateau-a-paris.fr, heading hierarchy polluee
- **Fichiers :** `src/data/posts*.json` (contenu HTML)
- **Effort :** 1h (script de nettoyage)
- **Action :** Supprimer le bloc boilerplate (H2+H3 commerciaux) des 11 posts concernes

---

### P1 — IMPORTANT (Impact SEO significatif, a faire sous 2 semaines)

#### 4. Ajouter breadcrumbs + schema BreadcrumbList sur les pages principales
- **Impact :** Navigation SEO manquante sur croisiere, galerie, faq, actualites, blog posts
- **Fichiers :** Croisiere.tsx, Galerie.tsx, FAQ.tsx, Actualites.tsx, ArticleDetail.tsx
- **Effort :** 3h
- **Note :** Les landing pages ont deja des breadcrumbs (bon modele a suivre)

#### 5. Ajouter og:locale:alternate sur toutes les pages
- **Impact :** Facebook/social ne connaissent pas les versions alternatives des pages
- **Fichiers :** `src/lib/metadata.ts` (getAlternates)
- **Effort :** 30min
- **Solution :** Ajouter `alternateLocales: locales.filter(l => l !== locale).map(getOgLocale)` dans openGraph

#### 6. Ajouter hreflang alternates dans le sitemap.xml
- **Impact :** Signal hreflang renforce pour Google (double signal : HTML head + sitemap)
- **Fichiers :** `src/app/sitemap.ts`
- **Effort :** 1h
- **Solution :** Utiliser la propriete `alternates.languages` de MetadataRoute.Sitemap

#### 7. Ajouter schemas JSON-LD manquants : WebSite + Organization
- **Impact :** Google Knowledge Panel, sitelinks search box
- **Fichiers :** `src/app/[locale]/layout.tsx` ou `page.tsx`
- **Effort :** 1h
- **Contenu :** WebSite (name, url, potentialAction SearchAction), Organization (name, logo, sameAs social)

#### 8. Ajouter /reservation dans le footer + 5 landings manquantes dans OccasionsGrid
- **Impact :** 5 landing pages a profondeur 2+ au lieu de 1
- **Fichiers :** `FooterVariants.tsx`, `OccasionsGrid.tsx`
- **Effort :** 1h
- **Landings manquantes :** anniversaire-mariage-seine, nouvel-an-seine, noel-seine, fete-des-meres-seine, seminaire-seine

## CE QUI FONCTIONNE BIEN

- **Meta tags** : title, description, og:*, twitter:card sur toutes les pages avec traductions
- **Hreflang** : implemente sur les 12 types de pages (hors blog slug issue)
- **x-default** : pointe vers /fr (locale par defaut)
- **next/image** : utilise partout avec formats avif/webp, priority sur LCP
- **JSON-LD** : LocalBusiness, Article, FAQPage, TouristTrip, TouristAttraction, BreadcrumbList (landings), Offer
- **SSG** : 356 pages pre-rendues, temps de build rapide
- **Security headers** : CSP, X-Frame-Options, HSTS, X-Content-Type-Options
- **Bundle** : leger (~50KB gzip), code splitting automatique
- **LCP** : blur placeholder + priority + fetchPriority="high" sur hero
- **Landing pages** : excellentes (breadcrumbs, schema, CTA, sticky bar mobile, related pages)
- **Traductions** : 480 cles x 6 locales, toutes completes
- **Footer** : liens crawlables avec `<Link>` i18n
- **Blog pagination** : tous les liens presents dans le HTML initial (SEO-friendly)
- **llms.txt** : present pour les LLMs
